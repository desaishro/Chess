# Chess Game – Two Player Web Chess Application

Chess Game is a browser-based chess application that allows two players to play chess against each other on the same device. The project implements official chess rules with an interactive and responsive interface, providing an engaging experience for casual and competitive players.

# 🚀 Key Features

### Two-Player Mode
Allows two players to play chess locally on the same device by taking alternate turns.

### Standard Chess Rules
Implements official chess rules including:
- Castling
- En passant
- Pawn promotion
- Check and checkmate detection

### Undo / Redo Moves
Players can undo or redo moves during the game, making it easier to correct mistakes.

### Responsive Design
The interface adapts to different screen sizes including:
- Desktop
- Tablet
- Mobile devices

### Minimalistic User Interface
Clean and simple interface designed for easy navigation and distraction-free gameplay.

# 🛠️ Technology Stack

## Frontend
HTML5 – Structure of the chessboard interface  
CSS3 – Styling and responsive design  
JavaScript – Game interaction and move handling  

## Libraries
Chess.js – Handles chess rules and game logic  
Chessboard.js – Renders the chessboard and manages piece movement

# 🎮 How to Play

### Start the Game
Open the project in your browser to display the chessboard.

### Select a Piece
Click on a chess piece to select it.

### Make a Move
Click on a valid square to move the selected piece.

### Invalid Moves
If a move is not valid according to chess rules, the piece will return to its original position.

### Game End Conditions
The game can end with:

- Checkmate
- Stalemate
- Draw
